# 🦆 Cryptoducks

Um aplicativo React criado com Vite, usado como projeto educacional para demonstrar conceitos de autorização e autenticação no front-end, no bootcamp de Desenvolvimento Web da TripleTen.

## 🛠️ Uso

Clone o repositório, instale as dependências e execute `npm run dev`.
O código inicial foi fornecido como base, e os commits representam as soluções de cada lição da Sprint.

## 🚀 Tecnologias
- React
- Vite

## 🎯 O que este projeto demonstra
- Fluxo de autenticação no front-end (registro, login e logout)
- Uso de tokens para manter a sessão do usuário
- Redirecionamento automático baseado no estado de autenticação
- Proteção de rotas usando React Router
- Uso da Context API para gerenciar autenticação globalmente

## 📦 Como instalar
```bash
git clone git@github.com:VanessaYuriAB/frontend-authorization-demo-pt.git
cd frontend-authorization-demo-pt
npm install
npm audit
npm run dev
```
